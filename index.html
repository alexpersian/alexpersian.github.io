Spotify Search

Search for artists, get album artwork, and listen to song clips.

New

The other day I was browsing API's and found out that Spotify has a public one. This sparked my interest in learning more about API's and handling server calls, so I thought up this web app to build.

It's built by leveraging the Spotify Web API. Along with a simple artist search, it displays album artwork, shows related artists, and you can click on an album to hear a song clip. Related artists are also clickable, and they will return the albums for that artist. This was a really fun app to build, and I hope that you have fun using it as well.
